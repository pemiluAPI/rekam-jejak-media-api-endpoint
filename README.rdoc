# Rekam Jejak Media EndPoint

[![Code Climate](https://codeclimate.com/github/pemiluAPI/rekam-jejak-media-endpoint.png)](https://codeclimate.com/github/pemiluAPI/rekam-jejak-media-endpoint)

## How to setup
1. `$ bundle install`
2. Configure your database credentials in database.yml
2. `$ bundle exec rake db:setup`
3. `$ bundle exec rails server`

## Data
[Rekam Jejak Media](https://github.com/pemiluAPI/pemilu-data/tree/master/rekam-jejak-media)
Run [Google Search](https://github.com/tj/google-search) `$ bundle exec rake run:gsearch`
